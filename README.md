# Uber Data Analysis Project 🚗

## Project Overview
This repository showcases data analysis for Uber ride data. It is structured as a portfolio project to present key insights regarding user trips, platform optimization, and business performance metrics.

## Tech Stack & Tools
* *Analysis Tool:* Power BI Desktop
* *Methodology:* Data Modeling, DAX Queries, Power Query (ETL)

## How to Review This Project
1. Download the Uber.zip.zip compressed archive file from the repository file list.
2. Extract the file locally on your system to access the repository contents.
3. Open the main file directly via *Power BI Desktop* to view and interact with the analytical dashboard components.
